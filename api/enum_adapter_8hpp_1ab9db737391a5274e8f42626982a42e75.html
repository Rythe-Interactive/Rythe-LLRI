Enum Documentation¶

enum llri::adapter_type¶

An informational enum describing the type of Adapter. The type does not directly affect how the related adapter operates, but it may correlate with performance or the availability of various features.

Values:

enumerator Other¶

The device type is not recognized as any of the other available types. This may for example be an APU, or a different form of supported processing unit.

enumerator Integrated¶

GPU embedded into the host CPU. Integrated GPUs typically have lower power cost but are (usually) less performant and share their memory with the host system.

enumerator Discrete¶

Separate GPU, usually connected to the host system through PCIe connectors.

enumerator Virtual¶

The adapter is a virtual node in a virtualization environment.

enumerator MaxEnum¶

The highest value in this enum.
